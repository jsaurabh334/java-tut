## 🛠️ How to Compile and Run

To compile and run any program from this collection:

### 1. Compile
Run `javac` with package path:
```bash
javac 01_Concrete_Class/Basic_ConcreteClass.java
```

### 2. Run
Specify fully qualified class path from the `Sample_Programs/` root:
```bash
java sample_programs.concrete_class.Basic_ConcreteClass
```

---

## 💡 Important Java Commands

* **`javac`**: Java Compiler. Compiles `.java` files into `.class` bytecode files.
* **`java`**: Java Virtual Machine (JVM) launcher. Executes Compiled bytecode.
* **`package`**: Groups related classes. It must mirror the directory path from the source root.
* **`import`**: Reuses classes from other packages.
